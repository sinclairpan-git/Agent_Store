# AI-SDLC v0.7.9 Release Notes

`v0.7.9` keeps the current startup path centered on `ai-sdlc init .`, which
runs the safe startup rehearsal before the user switches to AI chat.

## CLI Path

- If legacy instructions mention `ai-sdlc install`, the expected error is
  `No such command 'install'`.
- Use `ai-sdlc self-update check` for update discovery.
- Use `ai-sdlc self-update check --upgrade-existing` when an existing install
  should be upgraded.
- Release assets are published under `releases/download/v0.7.9`.

## Offline Assets

- Windows `.zip`: `ai-sdlc-offline-0.7.9-windows-amd64.zip`
