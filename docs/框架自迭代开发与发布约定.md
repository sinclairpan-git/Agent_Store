## 发布口径

- 普通用户主路径必须从安装后的可执行命令进入，而不是源码开发路径。
- 只有存在 live host evidence 时，才能声称宿主侧能力完成。
- 治理状态不得只停留在 materialized only；需要可验证加载或明确降级说明。
